
1. What is ASP.NET Framework?

--> ASP.NET MVC is a Web development framework from Microsoft

-------------------------------------------------------------------------------------------

2. What Is Wrong with ASP.NET Web Forms?

1. View State weight				- More request and response cycle
2. Page life cycle				- complicated life cycle 
3. False sense of separation of concerns	- difficult logic to view
4. Limited control over HTML 			- limited control
5. Leaky abstraction				- Hides html and http 
6. Low testability				- difficult unit testing

--> During development, you will find that the application you create is hard to test and hard to maintain.

-------------------------------------------------------------------------------------------

3. The New Features in MVC 5

--> Authentication Filters 
--> Filter Overrides 
--> Attribute Routing 

--> ASP.NET version 4.5.1, on which the MVC Framework 5 is built, has been enhanced as well. 
--> The most important change is the addition of the ASP.NET Identity API, which replaces the Membership system for managing user credentials.
