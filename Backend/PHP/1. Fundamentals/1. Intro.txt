1. What is PHP?

--> It is a server side scripting language and a powerful tool for making dynamic and interactive Web Pages

--> PHP stands for Hypertext Pre-precessor
--> It is Open source
--> It is used for server side logic.

-----------------------------------------------------------------------------------------

2. PHP Features

--> Cross Platform
--> Supports multiple server types
--> Easy to learn
--> Supports different types of DB

-----------------------------------------------------------------------------------------

3. Variable

--> It is a name given to a memory location which acts a container for storing data.
--> $age = 22;

-----------------------------------------------------------------------------------------

4. Constants

--> They are fixed values that do not change during execution time
--> define('PI', 3.14);

-----------------------------------------------------------------------------------------

5. Primitive Data Types

--> Integers
--> Float
--> Strings
--> Boolean
--> Undefined

-----------------------------------------------------------------------------------------

6. Reference Data Types

--> Arrays
--> Functions

-----------------------------------------------------------------------------------------

7. Arrays

--> An array is a data structure that contains a list of elements.
--> These elements are all of the same data type, such as integer or string.

--> $chocolates = array('Eclairs', 'Mars', 'Kitkat');

-----------------------------------------------------------------------------------------

8. Functions

--> It is a block of organised, reusable code that is used to perform a single, related action.
--> Two types: 1. Predefined functions and 2. User-defined functions
--> Syntax:

function addNumbers($num1, $num2){
	return $num1 + $num2;
}

-----------------------------------------------------------------------------------------

9. Conditional Statements

--> They are set of rules performed if a certain condition is met
--> It is like an 'If-Then' statement (If a condition is met, then an action is performed)
--> Syntax:

if(condition) {
   statement;
}

-----------------------------------------------------------------------------------------

10. Loops 

--> They are used to repeat a specific block until some end condition is met.
--> Three types: for, while, do-while
--> while(condition){
	loop code;
    }

--> for(begin; condition; step) {
	loop code;
    }
