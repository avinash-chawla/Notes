1. Create new project folder

--> Inside htdocs, create new folder 'tuts' (tutorial)
--> We will include all files inside this project folder

-----------------------------------------------------------------------------------------

2. index.php

--> Create new file inside project --> index.php
--> This will be html file only with php code tags inbetween.

--> PHP code tag: <?php ...  ?>

-----------------------------------------------------------------------------------------

3. Add path to php from environmental variables

--> C:\\xampp\\php\\php.exe

or

--> In vscode, preferences, settings
--> Add this line,

"php.validate.executablePath": "C:\\xampp\\php\\php.exe"

-----------------------------------------------------------------------------------------

4. Hello World

--> In index.php,

<?php 
    echo 'Hello World';
?>

-----------------------------------------------------------------------------------------

5. Php embedded in html tags

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First PHP Page</title>
</head>
<body>
    <h1><?php echo 'Hello World' ?></h1>
</body>
</html>
