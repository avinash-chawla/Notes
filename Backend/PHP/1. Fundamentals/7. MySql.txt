1. Setting up MySql Database

--> Open Xampp control panel --> start MySql --> Admin

--> Create new database --> ninja_pizza

--> Create table

--> id (int) (check auto-increment)
--> title, ingredients, email (varchar(255))
--> created_at --> current time stamp (set default current time stamp)

--> Insert data

title: ninja supreme
ingredients: tomato, cheese, fosu
email: ninja@gmail.com

--> Insert data

title: mario supreme
ingredients: tomato, cheese, mushroom
email: mario@gmail.com

-----------------------------------------------------------------------------------------

2. Connecting to the database

--> Create new user in MySql database
--> username: avinash
--> host: localhost
--> password : any
--> Check all permissions
--> Go

--> There are two ways, we can connect php with mySql
1. pdo  (php data objects)
2. sqli	(i stands for improved)

--> We will use sqli in this tutorial.

-----------------------------------------------------------------------------------------

--> In index.php,

<?php
    $conn = mysqli_connect('localhost', 'avinash', 'Avinash@3', 'ninja_pizza');

    if(!$conn){
        echo 'Connection error: ' . mysql_connect_error();
    }
?>

--> 'ninja_pizza' is our database name
--> $conn returns boolean value, true for successful connection
