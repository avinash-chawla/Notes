1. Installing PHP

--> There is a package Xampp 
--> It includes all technologies that we needed
--> PHP, Perl, Sql, MariaDB, Apache Server 

--> https://www.apachefriends.org/download.html

--> Select only those technologies in setup which you needed.

-----------------------------------------------------------------------------------------

2. Starting Apache Server

--> Open Xampp Control Panel
--> Start Apache

--> By default, it runs on port 80
--> Click on Admin to open browser with port 80 or head over to http://localhost/dashboard

-----------------------------------------------------------------------------------------

--> If you want to change it, click on config
--> Select first option --> Apache(httpd.conf)

--> A text file will get opened --> Change port number in Listen section
--> This problem arises when you have skype installed on your computer

-----------------------------------------------------------------------------------------

3. httpdocs

--> It is directory where project is served.
--> By default, apache looks for index file in httpdocs directory

--> There is a file index.php and includes code to redirect to dashboard/index.html
