--> ng new DatingApp-SPA  

-------------------------------------------------------------------------------------------------

2. Running Project

--> cd DatingApp-SPA
--> ng serve

-------------------------------------------------------------------------------------------------

3. Creating value component

--> ng g c value --skip-tests

-------------------------------------------------------------------------------------------------

4. Including HttpClientModule in our project

    BrowserModule,

5. Injecting HttpClient Service class into Value Component

export class ValueComponent implements OnInit {

  constructor(private http: HttpClient) { }


  ngOnInit(): void {
    this.http.get("https://localhost:44398/api/values")

-------------------------------------------------------------------------------------------------

9. value.component

export class ValueComponent implements OnInit {
  values : Object;

  constructor(private http: HttpClient) { }

    ngOnInit(): void {
    this.getValues();
  }

  getValues(){
    this.http.get<any[]>("https://localhost:44398/api/values")
    .subscribe((response) => {
      this.values = response;
      console.log(response);
    }, (error) => {
      console.log(error);
    });
  }

--> Best place to subscribe to observable is ngOnInit method

--> response is always observable object, thats y, values: Object 
--> Consider observable as stream of data

--> We are getting array of javascript objects, that y, get<any[]>

-------------------------------------------------------------------------------------------------

<table>

    </tr>
    <tr *ngFor="let value of values">

        <td>{{ value.name }}</td>
