1. What is Angular

--> It is Javascript Framework for building SPA

--> It is single Index page
--> Changes is done by manipulating dom elements by Angular

-----------------------------------------------------------------------------------

2. First App

--> Follow commands:

npm install -g @angular/cli
ng new my-app
cd my-app
ng serve

-----------------------------------------------------------------------------------

3. Basic files

--> e2e 		= end-to-end testing
--> node_modules 	= Dependencies of Angular
--> src 		= our application project


--> app.component.ts 	= Render as Javascript code inside script tag
--> app.component.html	= Content of our page


--> Angular is made up of small packages/modules 
--> Each package contain some features - eg. routing

--> Eg. To use two-way binding, we use ngModel directive, which is included in package - @angular/forms

--> app.module.ts	= Service class providing dependencies to our project

--> This is the module used in starting the application.

-----------------------------------------------------------------------------------

4. Global Styling

--> angular.json is the file for configuring our application

--> npm install --save bootstrap@3

--> It will add package in node_modules folder
--> It is available locally now

--> To make available globally, we need to add in angular.json file
--> stlyes: [
 	"node_modules/bootstrap/css/dist/bootstrap.min.css",
	"styles.css"
    ]

--> Recompile the Application after adding
