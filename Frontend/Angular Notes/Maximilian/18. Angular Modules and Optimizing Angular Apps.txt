1. What are Modules

--> They are ways of bundling Angular building blocks together.
--> Building blocks like components, services, directives and pipes

--> Note Points:

1. Angular analyzes NgModules to "understand" your application and its features
2. Angular modules define all building blocks your app uses
3. An app requires at least one module but may be split into multiple modules
4. Core angular features are included in Angular modules to load them only when needed
5. You cant' use a feature/building block without including it in a module

--------------------------------------------------------------------------------------------------

2. Analyzing the AppModule

--> One module does not communication with other modules
--> Component defined in one module, its selectors can not be used in other modules

--> We define other modules references in imports property
